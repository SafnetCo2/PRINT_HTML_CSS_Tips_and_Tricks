Print HTML CSS is a method of designing HTML content specifically for print or PDF output. It ensures that your documents â€” invoices, reports, catalogs, or blogs â€” look polished and professional when printed or converted to PDF.

Print CSS removes unnecessary web elements, optimizes layout, sets page sizes, and ensures fonts, colors, and images appear consistently across different printers and PDF tools.

1. Use Dedicated Print Stylesheets

Use @media print to control what appears in print. Hide navigation, footers, ads, and buttons, focusing only on essential content:

@media print {
  nav, footer, .ads { display: none; }
  article { width: 100%; font-size: 12pt; }
}

2. Set Up Page Breaks Properly

Control content flow between pages using:

h2 { break-before: page; }

This is important for multi-page documents like invoices, reports, or blogs.

3. Choose Print-Safe Fonts and Colors

Use system or web-safe fonts with high-contrast colors. Avoid solid black backgrounds as they can print unevenly and consume excessive ink.

4. Define Page Sizes and Margins

Use @page to set exact page size and margins:

@page {
  size: A4;
  margin: 1in;
}

This ensures content fits neatly on paper or in PDFs.

5. Test Across Browsers and PDF Tools

Test your print layout in Chrome, Firefox, Safari, and any HTML-to-PDF converters. Check image scaling, page breaks, and fonts to ensure consistency.
